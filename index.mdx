Lyria Studio is a desktop music tool that connects Google Lyria Realtime AI to a visual graph editor and multi-track timeline. Sketch out AI-generated instrumentals and vocals, capture the audio you like, arrange clips, and export a finished mix — all without leaving the app.

## How Lyria Studio works

Lyria Studio is built around two main workspaces: the **graph editor** and the **timeline**.

In the graph editor you connect nodes — prompts, instruments, vocals, and an output — to describe what the AI should generate. Weighted connections let you blend multiple influences. When you press **Go Live**, the graph drives a live Google Lyria Realtime session that streams audio continuously.

The timeline is where captured audio becomes a song. You can capture instrumental audio directly from the live stream, generate standalone vocals clips, and drop everything onto multi-track lanes. From there you trim, reorder, automate volume and pitch, and export to WAV or compressed audio.

<Steps>
  <Step title="Get a Gemini API key">
    Visit [Google AI Studio](https://aistudio.google.com/) and create a free API key. Lyria Studio uses this key to connect to Google Lyria Realtime.
  </Step>
  <Step title="Install and open Lyria Studio">
    Download the installer for your platform or clone the repo and run `npm run dev`. Paste your API key into the toolbar and click **Save Key**.
  </Step>
  <Step title="Build a graph and go live">
    Add a **Prompt** node, connect it to the **Output** node, type a musical idea, set your BPM, and press **Go Live** to start streaming AI-generated audio.
  </Step>
  <Step title="Capture, arrange, and export">
    Hit **Capture** to record the live stream onto the timeline, arrange your clips, tweak automation, and export the final mix as a WAV file.
  </Step>
</Steps>

<Note>
  Lyria Studio is currently in beta. Some workflows are still early-stage. If you run into anything unexpected, [open an issue on GitHub](https://github.com/AlanRoybal/lyria-studio/issues).
</Note>
